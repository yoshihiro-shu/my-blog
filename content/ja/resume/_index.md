## 職務要約

Webアプリケーションエンジニアとして、Go言語を中心としたバックエンド開発に約4年の実務経験を持ちます。株式会社ZUUにて金融メディアのフルスタック開発・スクラムマスター・データ基盤構築を経験した後、GMOペパボ株式会社にてゲームサーバープラットフォームおよびVPNサービスの開発・運用を担当。要件定義から設計・実装・リリースまでを一貫して遂行してきました。

直近ではエンジニア1名体制から3名体制へのチーム拡大を主導し、アジャイル開発体制の構築・オンボーディング・プロジェクトマネジメントも担当。前職でもスクラム導入やチームビルディングを経験しており、技術力とチームビルディングの両面から事業成長に貢献してきました。

## スキルと経験

**バックエンド開発**: Go言語をメインに、connect-go / gRPC、echo、REST API設計、バッチ処理など幅広いバックエンド技術を実務で活用。Python（Django REST Framework、Pandas、dbt）やTypeScript / React / Next.jsによるフルスタック開発も経験。データ基盤構築ではBigQuery・Cloud Functions・Cloud Run Jobsを活用し、Kubernetes上でのアプリケーションデプロイやGitHub Actions / Argo CDによるCI/CD改善も業務で経験。

**アーキテクチャ設計**: DDDに基づくリアーキテクチャの設計・実装、決済基盤設計（クレジットカード・請求書払い・PayPay）、マイクロサービス間通信設計（connect-go / gRPC / 3プロトコル対応）を推進。テスト戦略設計（単体テスト / E2E / シナリオテスト / 負荷テスト）やセキュリティ対策（CSRF対策、OWASP ZAP脆弱性診断、認証設計）にも取り組む。

**チームリーダーシップ**: スクラムマスターとしてアジャイル開発を導入・定着させた経験を持つ。オンボーディング設計・実施（新卒・業務委託・復帰メンバー）、チームビルディング（インセプションデッキ・スキルマップ）、ふりかえりファシリテーションなどを通じ、チームの生産性と心理的安全性を向上。

**ビジネスインパクト**: 技術的な意思決定をする際、事業への影響を常に意識している。決済基盤やデータ基盤の設計では、経理・営業・サポートなど非エンジニア部門と直接やりとりしながら業務フローを整理し、技術と業務の両面から課題を解決してきた。直近では競合サービスの技術的な調査を行い、プロダクトのロードマップ提案にも取り組んでいる。
